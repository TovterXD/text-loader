# Веб-сборщик контента с внешних сайтов

Приложение для автоматического сбора HTML-страниц по заданным темам с отображением хода загрузки и возможностью просматривать сохранённые данные без доступа к интернету.

---

## Возможности

- **Поиск по темам:** На сервере задан набор ключевых слов (`news`, `code`, `search`) для выборки контента.

- **Индикация прогресса в реальном времени:** Через технологию **Server-Sent Events (SSE)** передаётся информация о загруженных байтах и проценте выполнения задачи.

- **Локальное хранилище (LocalStorage):** Все загруженные данные сохраняются в браузере и доступны для просмотра в отдельном `iframe` — без повторных запросов к серверу и без подключения к сети.

- **Обработка ошибок:** Предусмотрены сценарии с некорректным запросом, потерей соединения или превышением лимитов хранилища.

---

## Используемые технологии

- **Сервер:** Node.js, Express, Axios  
- **Клиент:** Чистый HTML5, CSS3, JavaScript (Fetch, SSE, LocalStorage)

## Запуск проекта

1. **Скопируйте репозиторий:**
   ```bash
   git clone <ссылка_на_репозиторий>
   cd text-loader
2. **Установите зависимости:**
   ```bash
   npm install
   ```
3. **Запустите сервер:**
   ```bash
   node .
   ```
4. **Откройте приложение в браузере:** [http://localhost:3000](http://localhost:3000)
